# Qt程序

本程序主要为正运动控制器上位机，通过Ether CAT总线和控制器通信
# 开发环境
- 操作系统：Windows 10 X64 22H2
- 开发环境：Qt Creater、Visual Stduio Code
- 编译环境：qt5.15.2 MinGW 64bit
# 功能说明
参数设置
- 控制器连接，可以选择需要连接的IP地址，并且实时显示当前所处的模式：实机模式或是仿真模式
- 可以通过参数配置界面调整每个轴的参数如，脉冲当量，正负限位，，还可以调整加工速度。
自动加工
- 对生成好的G代码进行解析，并根据G代码进行雕刻加工，具有对刀点记忆，对刀点恢复等功能。
手动控制
- 通过鼠标点击不同的按键实现轴移动，加入键盘联动功能。
